(static) collect(htmlElementopt, patternopt, customCollectorsopt)

发起渲染函数。收集总入口。设置好数据路径(html attribute data)和渲染器(html attribute renderer)后，调用该函数，从所有相关元素收集录入信息到根数据。
也可使用封装了该函数的 jQuery 函数:

$(htmlElement).collect(pattern, customCollectors)
Parameters:
Name Type Attributes Default Description
htmlElement HTMLElement <optional>
 document.body

html 元素，收集该元素及其子元素。
pattern object | Array.<object> <optional>

命中的数据，可提供数组。提供该参数后，只有展开数据中包含有 pattern 的对象时才发生收集。
customCollectors object <optional>

自定义收集器（含管道）。如

{
    myCollector : function(el, value){return el.innerHTML}
}

对 htmlElement 及其子元素有指定 collector="myCollector"，该函数会被应用。需要说明的是，在下次收集时不再需要提供 myCollector 函数。

(static) render(htmlElementopt, patternopt, customRendersopt)

发起渲染函数。渲染总入口。设置好数据路径(html attribute data)和渲染器(html attribute renderer)后，调用该函数，绘制所有相关元素。
也可使用封装了该函数的 jQuery 函数:

$(htmlElement).render(pattern, customRenders)
Parameters:
Name Type Attributes Default Description
htmlElement HTMLElement <optional>
 document.body

html 元素，渲染该元素及其子元素。
pattern object | Array.<object> <optional>

命中的数据，可提供数组。提供该参数后，只有展开数据中包含有 pattern 的对象时才发生渲染。
customRenders object <optional>

自定义渲染器（含管道）。如

{
    myRender : function(el, value){el.innerHTML = value}
}

对 htmlElement 及其子元素有指定 renderer="myRender"，该函数会被应用。需要说明的是，在下次渲染时不再需要提供 myRender 函数。
