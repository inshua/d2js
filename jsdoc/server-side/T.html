T

new T()

服务器端类型转换器，如转换失败则抛错，这组工具会修改 record 的值，同时具备转换和校验的功能。

使用方法:

$V(rcd,{ 
            name  : [T.string, V.notNull, V.shortest(5), V.unique('table')],
            qq      : T.number(20),
            email : [V.notNull, V.email],
            telphones : [T.array],
            tag : [T.object, T.json],        // 先将 object 从字符串转为 js 对象，再加上 $JSON() 引用 
      })
